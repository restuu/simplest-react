# simplest-react
Create and configure React project from scratch without `create-react-app`.

## Tree
```
.
├── README.md
├── index.html
├── package-lock.json
├── package.json
├── src
│   ├── components
│   │   └── Home.jsx
│   └── main.js
└── webpack.config.js
```

## Quickstart
```bash
npm install
npm run build
npm start
```
